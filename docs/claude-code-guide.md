# Claude Code 完全ガイド（初心者向け）

> Claude Code を初めて使う人のための、インストールから活用までの完全ガイドです。

---

## Claude Code とは

Claude Code は、Anthropic 社が提供する **AI コーディングアシスタント** です。
ターミナル（黒い画面）やエディタから直接 AI と対話して、以下のようなことができます：

- コードの意味を説明してもらう
- プログラミングの概念を質問する
- コードを生成・修正してもらう
- エラーの原因を調べてもらう
- ファイルの読み書きを任せる
- テストやレビューを依頼する

本プロジェクトでは、Claude Code を **学習パートナー** として使います。

## 料金プラン

Claude Code を使うには、有料プランへの加入が必要です。

### 個人向けプラン

| プラン | 月額 | Claude Code | おすすめ度 | 説明 |
|--------|------|-------------|-----------|------|
| **Free** | 無料 | ❌ 使えない | - | チャットのみ |
| **Pro** | $20/月 | ✅ 使える | ⭐ 学習に最適 | 個人利用に十分な使用量 |
| **Max 5x** | $100/月 | ✅ 使える | - | Pro の 5 倍の使用量 |
| **Max 20x** | $200/月 | ✅ 使える | - | Pro の 20 倍の使用量 |

### チーム・企業向けプラン

| プラン | 月額 | Claude Code |
|--------|------|-------------|
| **Team** | $25〜30/人 | ✅ 使える |
| **Enterprise** | 要問合せ | ✅ 使える |

### どれを選べばいい？

**結論：まずは Pro（$20/月）で始めましょう。**

- 学習用途なら Pro で十分です
- 使用量の上限に頻繁に達するようなら Max を検討
- 支払いはクレジットカードで、いつでも解約可能です

### アカウント作成手順

1. https://claude.ai にアクセス
2. 「Sign up」でアカウントを作成（Google アカウントまたはメールアドレス）
3. ログイン後、左下の「Upgrade to Pro」をクリック
4. 支払い情報を入力して Pro プランに加入

## インストール方法

Claude Code には複数の利用方法があります。**本プロジェクトではターミナル CLI を推奨**しますが、好みに合わせて選んでください。

### 方法 1: ターミナル CLI（推奨 ⭐）

最も多機能で、本プロジェクトの学習プロトコルに最適です。

#### macOS の場合

```bash
# ターミナル.app を開いて以下を実行
curl -fsSL https://claude.ai/install.sh | bash
```

#### Windows の場合

```powershell
# PowerShell を開いて以下を実行
irm https://claude.ai/install.ps1 | iex
```

#### インストール確認

```bash
claude --version
```

バージョン番号が表示されれば成功です！

#### 初回ログイン

```bash
claude
```

初めて実行すると、ブラウザが開いて認証画面が表示されます。
claude.ai のアカウントでログインすれば完了です。

---

### 方法 2: VS Code 拡張機能

コードエディタの中で直接 Claude Code を使いたい場合。

1. VS Code を開く
2. 左サイドバーの拡張機能アイコン（四角が4つ）をクリック
3. 検索バーに「Claude Code」と入力
4. 「Claude Code」（Anthropic 公式）をインストール
5. VS Code を再起動
6. サイドバーに Claude Code のアイコンが表示される

**必要条件**: VS Code バージョン 1.98.0 以上

---

### 方法 3: Claude Desktop アプリ

ターミナルが苦手な人向け。グラフィカルな画面で操作できます。

1. https://claude.ai/download からダウンロード
   - macOS: `.dmg` ファイル
   - Windows: `.exe` ファイル
2. インストールして起動
3. Anthropic アカウントでログイン
4. 「Code」タブを選択
5. プロジェクトフォルダを指定

---

### 方法 4: Web アプリ（claude.ai）

インストール不要。ブラウザから直接使えます。

1. https://claude.ai にアクセス
2. ログイン
3. 「Code」モードを選択（画面上部）

※ ファイル操作やコマンド実行は制限される場合があります。

```bash
claude -r
```

## スラッシュコマンド

Claude Code のセッション中に使える便利なコマンドです。先頭に `/` をつけて入力します。

### よく使うコマンド

| コマンド | 説明 | 使いどころ |
|----------|------|-----------|
| `/help` | ヘルプを表示 | 困ったとき |
| `/clear` | 会話履歴をクリア | 話題を切り替えたいとき |
| `/compact` | 会話を要約して圧縮 | 会話が長くなったとき |
| `/cost` | 今の会話のトークン使用量を表示 | 使用量が気になるとき |
| `/init` | CLAUDE.md を自動生成 | プロジェクト初期設定 |

### その他のコマンド

| コマンド | 説明 |
|----------|------|
| `/config` | 設定を変更 |
| `/login` | アカウント切り替え |
| `/doctor` | インストール状態の診断 |
| `/memory` | メモリ（自動記憶）の確認 |
| `/review` | コードレビューを依頼 |
| `/pr-comments` | PR のコメントを確認 |

## 権限（Permissions）の仕組み

Claude Code が何かを実行しようとするとき、あなたに許可を求めます。
これは安全のための仕組みです。

### 許可の種類

| 選択肢 | 意味 |
|--------|------|
| **Allow（許可）** | 今回だけ許可する |
| **Allow always（常に許可）** | このプロジェクトで常に許可する |
| **Deny（拒否）** | 実行しない |

### 権限モード

セッション中に `Shift + Tab` で切り替えられます：

| モード | 説明 | おすすめ |
|--------|------|---------|
| **default** | 毎回許可を求める | ⭐ 初心者はこれ |
| **acceptEdits** | ファイル編集は自動許可 | 慣れてきたら |
| **plan** | 編集前に計画を提示 | 大きな変更のとき |
| **auto** | ほぼ自動で実行 | 上級者向け |

**初心者の方は default モードのまま使いましょう。** 一つずつ確認しながら進められて安心です。

## CLAUDE.md とは

プロジェクトのルートに置く設定ファイルです。Claude Code がプロジェクトで作業するときに自動的に読み込みます。

本プロジェクトにはすでに `CLAUDE.md` が用意されているので、追加設定は不要です。

### CLAUDE.md に書いてあること

- プロジェクトの概要（何をするプロジェクトか）
- ビルド・実行コマンド
- コーディング規約
- 学習支援時の注意事項

### 自分で CLAUDE.md を作りたい場合

```bash
# Claude Code に自動生成してもらう
/init
```

## このプロジェクトでの Claude Code の使い方

### 前半セッション（Session 1〜10）: チューターモード

プログラミングの基礎を学ぶフェーズでは、Claude Code を **先生** として使います：

```
# 概念を質問する
「変数」と「定数」の違いを初心者にわかるように教えて

# コードの意味を聞く
このコードが何をしているか1行ずつ説明して
```

```
# 理解度テストを出してもらう
今日学んだ HTML の基礎について、3問の選択問題を作って

# 学習記録を作ってもらう
今日の学習内容をまとめて knowledge/sessions/ に保存して
```

> ⚡ この段階では Claude Code にコードを書かせるのではなく、
> **あなたが理解すること** に集中しましょう。

### 後半セッション（Session 11〜15）: 実装パートナーモード

実際にアプリを作るフェーズでは、Claude Code を **開発パートナー** として使います：

```
# コードを生成してもらう
Supabase でログイン機能を実装して、各ファイルの説明もつけて

# コードレビューを依頼
今回変更したファイルをレビューして、改善点を教えて

# エラーを解決してもらう
このエラーの原因と修正方法を教えて
```

## よくあるトラブルと対処法

### 「claude: command not found」と出る

インストールが完了していないか、パスが通っていません：

```bash
# 再インストール
curl -fsSL https://claude.ai/install.sh | bash

# ターミナルを再起動してから再度試す
claude --version
```

### ログインできない

```bash
# 認証をやり直す
claude login
```

ブラウザが開くので、claude.ai のアカウントでログインし直してください。

### 「Rate limit exceeded」と出る

使用量の上限に達しました。以下の対処法があります：

- しばらく時間をおいてから再開（通常数時間でリセット）
- `/compact` で会話を圧縮して使用量を節約
- Max プランへのアップグレードを検討

### 動作がおかしいとき

```bash
# 診断コマンドを実行
claude /doctor
```

問題を自動で検出して対処法を教えてくれます。

### ファイルの変更が意図と違った

Claude Code が提案する変更は、許可する前に内容を確認できます。
間違った変更を許可してしまった場合：

```bash
# Git で変更を元に戻す（Claude Code の中で）
さっきの変更を元に戻して
```
