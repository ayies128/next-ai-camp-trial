# Phase 4: 理解チェックプロンプト（Claude Code 用）

## 使い方

Phase 3（実践）が終わったら、以下のプロンプトで理解度を確認してください。
Claude Code がセッションの会話内容をもとに問題を作ります。

## プロンプト

```
今日の学習内容をもとに、理解度チェックをして。

ルール:
- 1問ずつ出題して、答えたら次の問題を出して
- 選択問題 15〜20 問 ＋ 記述問題 3〜5 問、たっぷり出して
- 同じ概念でも角度を変えて繰り返し出す（定義→具体例→間違い探し→使い分けなど）
- 選択問題は AskUserQuestion ツールを使って、クリックで選べるようにして
- 記述問題も AskUserQuestion ツールで質問して
- 各問の正解・不正解のあとに簡単な解説をつけて
- 全問終わったら結果をまとめて
```

## 補足

- 間違えた問題があれば、そのまま Claude Code に追加で説明を求められます
- 「もう少し難しい問題を出して」「別の角度から出題して」などのリクエストもOK
