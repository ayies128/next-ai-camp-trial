# Phase 2: 教材生成プロンプト（Claude Code 用）

## 使い方

Phase 1 でテーマが決まったら、以下のプロンプトを入力してください。
`{テーマ名}` の部分を今日のテーマに置き換えます。

## プロンプト

```
今日のテーマは「{テーマ名}」です。
初心者向けの教材を作って。
構成: 理論説明 → コード例 → 実践課題（1〜2個）
60分以内で完了する分量で、Next.js の App Router 前提で解説して。
```

## 補足

- 前半セッション（1-10）: 概念説明を重視してもらうよう追加で依頼してもOK
- 後半セッション（11-15）: 実装手順を重視してもらうよう追加で依頼してもOK
