# Phase 5: 学習まとめプロンプト（Claude Code 用）

## 使い方

理解チェックが終わったら、以下のプロンプトで学習記録を作成・保存してもらいます。
Claude Code がファイルに直接書き出すので、コピペ不要です。

## プロンプト

```
今日のセッション内容をまとめて、学習記録を作成して
knowledge/sessions/ に保存して。
ファイル名は {YYYY-MM-DD}-session-{番号}-{テーマ}.md で。
見出しは「今日のテーマ」「やったこと」「学んだこと」「理解できていないこと」「次やること」で。
コードブロックも適切に整形して。
```

## 補足

- 保存後、Obsidian で開いて内容を確認・加筆してください
- 「理解できていないこと」が次回の復習ポイントになります
