# Phase 1: テーマ設定プロンプト（Claude Code 用）

## 使い方

Claude Code のセッションで以下をそのまま入力してください。
Claude Code が `docs/roadmap.md` と `knowledge/sessions/` を自動で読み取って、最適なテーマを提案します。

## プロンプト

```
docs/roadmap.md と knowledge/sessions/ を読んで、
次に学ぶべきセッションのテーマを提案して。
30〜60分で完了する粒度で、以下を含めて：
- 目的
- 事前知識（前回までに学んだこと）
- 到達点
- ミニ課題（1〜2個）
```
