# AIブートキャンプ 体験版（Session 0〜3）

> Claude Code をパートナーに、プログラミングの第一歩を踏み出そう。

## これは何？

現役ベンチャー企業CTOが作った、**プログラミング完全未経験者向けの教材**の体験版です。

AI（Claude Code）が先生役になって、あなたのペースに合わせて教えてくれます。
何度聞いても嫌な顔をされません。24時間いつでも質問できます。

この体験版では、全20セッションのうち**最初の4セッション**を無料で体験できます。

## 体験版で学べること

| Session | テーマ | 学ぶこと |
|---------|--------|---------|
| 0 | プロジェクトの歩き方 | ツールのセットアップ、学習の進め方 |
| 1 | Git ってなに？ | ファイルの変更を記録する方法（セーブポイント） |
| 2 | GitHub を使ってみよう | ブランチとPR（チーム開発の基本） |
| 3 | Web の世界を知ろう | HTML/CSS/JavaScript の役割、開発者ツール |

## はじめかた

### 必要なもの
- PC（Mac または Windows）
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code/overview)（月額約$20）

### セットアップ手順

1. このリポジトリを自分のアカウントにフォーク（またはクローン）
```bash
git clone https://github.com/ennagara128/next-ai-camp-trial.git
cd next-ai-camp-trial
```

2. Claude Code を起動
```bash
claude
```

3. Claude Code に聞いてみよう
```
このプロジェクトのREADMEを読んで、どういう学習プロジェクトなのか教えて
```

4. 詳しいセットアップは [docs/setup.md](docs/setup.md) を参照

### 学習の進め方

毎回のセッションは [学習プロトコル](docs/learning-protocol.md) に沿って進めます。
Claude Code の使い方は [Claude Code ガイド](docs/claude-code-guide.md) を参照してください。

この体験版で「AIと一緒にプログラミングを学ぶ体験」を気に入っていただけたら、
